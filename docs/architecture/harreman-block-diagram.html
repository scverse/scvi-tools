External model · DestVI

DestVI

Spatial deconvolution. Provides per-spot cell-type proportion matrix via get_proportions(). Each cell type becomes a scaled count layer in AnnData.

get_proportions() mode="cell_type" layers[cell_type]
External model · ResolVI

ResolVI

Segmentation-error denoising. Provides denoised expression via get_normalized_expression(), attached as layers["denoised"].

get_normalized_expression() layers["denoised"]
Core model · SCVIVA

SCVIVA

Niche-aware variational inference. Provides latent representation via get_latent_representation() used to construct the spatial KNN graph.

get_latent_representation() obsm["X_scVIVA"] KNN coords

Orchestrator · scvi.external.harreman

HarremanAnalysis

Central analysis class for spatial metabolic cell-cell communication (CCC). Wraps hotspot, tools, preprocessing, and plots behind a unified ordered-step interface. State is persisted in adata.uns["harreman"]. Model integration extracts relevant outputs at __init__ time — no stale model references.

setup() filter_genes() compute_gene_pairs() compute_cell_communication() compute_interacting_cell_scores() select_significant_interactions() ha.hs ha.tl ha.pl adata.uns["harreman"] HarremanResults

Local autocorrelation and local correlation statistics on the spatial KNN graph. Identifies spatially variable genes and co-varying gene pairs. Foundation for CCC inference.

Analysis data flow

Inputs

AnnData + HarremanDB

Raw counts, spatial coordinates, ligand-receptor / metabolite database (HarremanDB).

Register

setup_anndata()

pp.setup_anndata() registers count layer, cell-type key, species. Writes uns["harreman"] namespace.

Autocorrelation

hotspot statistics

filter_genes() → compute_gene_pairs() via local autocorrelation + local correlation on KNN graph.

CCC inference

Communication scores

compute_cell_communication() + compute_interacting_cell_scores() → per-cell and cell-type interaction matrices.

Outputs

Results + plots

adata.uns["harreman"] · HarremanResults · correlation heatmaps · spatial interaction maps.
